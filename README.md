# 🎓 Student Management System

A simple and interactive **Student Management System** developed using **HTML, CSS, and JavaScript**. This project allows users to manage student records through a clean and user-friendly interface.

## 📖 Overview

The Student Management System helps users:
- Add student details
- View student records
- Manage student information easily
- Perform operations through a responsive interface

This project is built as a frontend application to demonstrate JavaScript DOM manipulation and responsive web development.

---

## ✨ Features

✅ Add Student Details  
✅ Display Student Records in Table Format  
✅ Responsive User Interface  
✅ User-Friendly Layout  
✅ Dynamic Data Handling using JavaScript

3. Right click → **Open with Live Server**

